# RAG Components

Contains components for Retrieval-Augmented Generation and intent clarification.

## Files
- `ambiguity_resolver.py`: Clarifies ambiguous user intents.
- `dietary_enforcer.py`: Ensures dietary restrictions are respected.
- `prompt_builder.py`: Constructs prompts for the LLM.
- `temporal_resolver.py`: Resolves time-related queries.
