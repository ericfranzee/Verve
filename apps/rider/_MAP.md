# apps/rider/ - Map

## Purpose
Root directory for the Verve Rider Pulse App. This lightweight app handles dispatch, navigation, and earnings for Nexus delivery riders.

## Files
- `pubspec.yaml`: Flutter project configuration.
- `lib/`: Main source code directory.
- `test/`: Test directory.

## Constraints
- Minimal feature set, robust functionality (Bio-Handshake, SOS).
