# Verve Mono-repo Structure

This repository contains the full stack for the Verve Cognitive Commerce platform.

- `apps/mobile`: Flutter consumer app.
- `apps/rider`: Flutter rider pulse app.
- `packages/core`: Shared libraries and types.
- `services/orchestrator`: Go middleware orchestration.
- `services/intent-engine`: Python FastAPI intent and inference engine.
- `services/nexus-api`: Node.js GraphQL Nexus API.
- `docs/`: Technical specifications and plans.
